# Staqq

Staqq is an iOS application designed to help you organize and manage school documents, flyers, and other papers by digitizing them through scanning and OCR.

## Features

- **Document Scanning**: Easily scan multi-page documents with automatic perspective correction.
- **Auto/Manual Capture**: Toggle between automatic document detection or manual shutter.
- **AI-Powered OCR**: Automatically extracts titles and event dates from scanned images.
- **Smart Organization**: Automatically categorizes documents by deadlines and custom folders (tags).
- **Calendar Integration**: Add document deadlines directly to your iOS Calendar.
- **Handwritten Notes**: Annotate your scans using Apple Pencil or touch via PencilKit.
- **PDF Export**: Share your digitized documents as PDF files.

## Tech Stack

- **SwiftUI**: Modern declarative UI.
- **SwiftData**: Robust local data persistence.
- **Vision & VisionKit**: High-performance document scanning and text recognition.
- **PencilKit**: Smooth handwriting and drawing interface.
- **EventKit**: Seamless calendar integration.
- **ImageIO**: Optimized image handling and thumbnail generation.

## Getting Started

1. Open `Staqq.xcodeproj` in Xcode 15 or later.
2. Select your target device or simulator.
3. Build and run the project.

## Requirements

- iOS 17.0+
